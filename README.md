## 📋 Описание проекта

**DataMarker** — это универсальное расширение для браузера, которое превращает любой веб-сайт в источник данных для машинного обучения и аналитики. Оно позволяет собирать текстовые данные и создавать размеченные датасеты прямо в браузере.

### 🎯 Проблема, которую решает проект

При подготовке данных для ML-моделей часто возникает задача сбора и разметки данных из открытых источников. Существующие решения либо требуют программирования (парсинг), либо неудобны для быстрой разметки. DataMarker объединяет оба процесса в одном инструменте.

### ✨ Возможности

- **Сбор данных**: выделяйте текст, собирайте все ссылки или весь текст страницы
- **Разметка данных**: кликайте на элементы и присваивайте им метки (Positive/Neutral/Negative/Custom)
- **Контекстное меню**: быстрый сбор через правый клик мыши
- **Хранение**: все данные сохраняются локально в браузере
- **Экспорт**: выгрузка в CSV и JSON для дальнейшего использования
- **Интуитивный интерфейс**: понятный UI с подсветкой элементов

## 🛠 Технологический стек

- **Manifest V3** - современная архитектура расширений Chrome
- **HTML5/CSS3** - адаптивный интерфейс
- **JavaScript (ES6+)** - вся логика расширения
- **Chrome Storage API** - локальное хранение данных
- **Chrome Downloads API** - экспорт файлов
- **Chrome ContextMenus API** - контекстное меню

## 📥 Инструкция по установке (Chromium-браузеры)

Данная инструкция является универсальной и подходит для **Google Chrome, Яндекс.Браузера, Microsoft Edge, Opera и Brave**.

1.  **Подготовьте файлы** 📁
    * Убедитесь, что все файлы расширения (`manifest.json`, `background.js`, `popup.html` и др.) находятся в одной локальной папке на вашем компьютере (например, в папке `focus-mode`).

2.  **Откройте страницу расширений** 🌐
    * Введите в адресную строку браузера: 
      * Для Chrome/Brave/Yandex: `chrome://extensions/`
      * Для Microsoft Edge: `edge://extensions/`

3.  **Включите режим разработчика** 🛠️
    * Найдите переключатель **Developer mode** (Режим разработчика) в правом верхнем углу и активируйте его.

4.  **Загрузите расширение** 📤
    * Нажмите на появившуюся кнопку **Load unpacked** (Загрузить распакованное расширение).
    * В системном диалоговом окне выберите папку `focus-mode`, в которой лежат файлы проекта.

5.  **Закрепите для быстрого доступа** 📌
    * Нажмите на иконку «пазла» (Расширения) в панели инструментов браузера.
    * Найдите в списке **Focus Mode** и нажмите на иконку булавки (**Pin**), чтобы таймер всегда был на виду.
